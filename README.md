# Smart Traffic Management System
# Introduction
The Smart Traffic Management System is a Python-based project designed to reduce urban traffic congestion using intelligent traffic signal control. 
Unlike traditional fixed-time traffic lights, this system dynamically changes signal timing according to traffic density and emergency situations.
# Objective
Reduce traffic congestion
Minimize waiting time
Improve traffic flow
Give priority to emergency vehicles
Simulate smart city traffic control
# Technologies Used
Python
Random Module
Time Module
Object-Oriented Programming (OOP)
# Working
The system monitors traffic on four roads:
North
South
East
West
Vehicle counts are generated dynamically.
The road with the highest traffic gets the green signal.
Green signal timing changes automatically based on traffic density.
If an emergency vehicle (ambulance/fire truck) is detected, that road gets immediate priority.
# Features
Adaptive traffic signal timing
Traffic density monitoring
Emergency vehicle priority
Real-time traffic simulation
Intelligent decision-making
